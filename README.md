# Food Expiry Reminder System

## 📌 Description
A Java AWT-based desktop application to manage and track food expiry dates.

## 🚀 Features
- Add food items with expiry dates  
- Update and delete entries  
- User-friendly GUI using AWT  
- Event handling using ActionListener  

## 🛠️ Tech Stack
- Java  
- AWT (GUI)  

## 💡 Purpose
Helps users manage food efficiently and reduce waste by tracking expiry dates.
